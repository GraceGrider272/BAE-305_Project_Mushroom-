# BAE-305_Project_Mushroom-

## Final Objectives:

User capability 1: Users can select media ratios. [Time Required – 5]

User capability 2: The user can observe the system automatically dispense the selected
ratios of media. [Time Required – 5]

User capability 3: The user can start the mixing process with a control mechanism. [Time
Required – 3]

User capability 4: Users can easily disassemble key components for cleaning. [Time
Required – 3]

User capability 5: The user will be alerted to an empty hopper by an indicator. [Time
Required – 3]


## Design Alternatives:

**Design Alternative 1:**
- Build a mixer that calculates ratios based on weight balance.

Justification 1:
- This option fulfills the project description by providing an alternative method for
ratio calculation and sensing.

**Design Alternative 2:**
- Build a media mixer controlled by an app.

Justification 2:
- This design choice meets the criteria and enables the project to be remotely
controlled rather than requiring manual control operations.

**Design Alternative 3:**
- The volume is measured by a time-of-flight laser sensor in the hoppers.

Justification 3:
- This still meets the criteria because the area of the hoppers does not change,
thereby allowing proper measurement of the media.
